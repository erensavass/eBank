## Overview

eBank is a native iOS mobile banking application developed with **SwiftUI** as part of my software engineering internship.

The project replicates the core experience of a modern digital banking application, providing an intuitive interface for account management, money transfers, transaction history, and user authentication while following Apple's native development practices.

---

## Features

- 🔐 Secure user authentication
- 💳 Digital banking dashboard
- 💰 Account balance management
- 🔄 Money transfer workflow
- 📄 Transaction history
- 👤 User profile management
- 🌙 Modern SwiftUI interface
- 📱 Native iOS experience
- 🌐 RESTful API integration
- 🏗️ MVVM architecture
